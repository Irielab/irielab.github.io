# 入江研究室 Web ページ

## ファイル構成
``` 
web-lab-repository/
│── public/                # 静的ファイル (画像・CSS・JS)
│   ├── images/            # 研究室の画像・ロゴ
│   ├── css/               # スタイルシート
│   ├── js/                # JavaScript ファイル
│── src/                   # メインのソースコード
│   ├── index.html         # ホームページの HTML
│   ├── about.html         # 研究室紹介ページ
│   ├── members.html       # メンバー紹介ページ
│── docs/                  # ドキュメント関連
│── README.md              # このファイル
│── .gitignore             # Git 無視ファイル
│── package.json           # npm 依存関係 (必要なら)
│── server.js              # ローカルテスト用サーバー (Node.js)
```
